**Author:** StopCry

## Overview

This repository contains technical documentation for a community-tested solution addressing an issue where **STP_Carryable_Log** objects fall through the terrain after harvesting trees while using **Gaia Pro VS** together with **Survival Template Pro Multiplayer (Netick)**.

The repository includes:

* 📄 Technical documentation (PDF)
* 🎥 Complete video walkthrough

---

## Important

This is **not an official fix** from **XWizard Games** or **Polymind Games**.

It is an independent community contribution that has been tested with the project configuration described in the documentation.

This repository does **not** contain or redistribute any proprietary assets or source code. It only provides technical documentation and minimal code snippets required to reproduce the solution.
